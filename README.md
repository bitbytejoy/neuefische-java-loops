# Wo ist die 1?

Schreib ein Programm, dass uns sagt an welcher Stelle die Nummer 1 in einer Reihe von Nummer vorkommt!

Eingang Beispiel:

    int[] numbers = new int[]{ 1, 2, 3, 4, 5 };

Ausgang Beispiel:

    0

---

Eingang Beispiel:

    int[] numbers = new int[]{ 2, 3, 4, 1, 5 };

Ausgang Beispiel:

    3

---

Eingang Beispiel:

    int[] numbers = new int[]{ 2, 3, 4, 5 };

Ausgang Beispiel:

    -1
