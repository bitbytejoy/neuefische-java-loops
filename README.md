# 1 Plus?

Schreib ein Programm, dass 1 zu jeder Nummer in einer Reihe addiert!

Eingang Beispiel:

    int[] numbers = new int[]{ 1, 2, 3, 4, 5 };

Ausgang Beispiel:

    2
    3
    4
    5
    6

---

Eingang Beispiel:

    int[] numbers = new int[]{ 3, 4, 1, 5 };

Ausgang Beispiel:

    4
    5
    2
    6
